Historia y Evolución del Diseño Web

El Comienzo: 1991-1993 Diseño web con texto

el nacimiento del reciente estándar del lenguaje HTML, hacía que las posibilidades gráficas o multimediales de las páginas web fueran muy limitadas. Existían pocas etiquetas HTML que sólo permitían trabajar con texto e hipertexto, entre otros.La gráfica de esta época corresponde sólo a texto. Los botones están hechos con hipervínculos.

HTML

En HTML, siglas de HyperText Markup Language («lenguaje de marcas de hipertexto»), hace referencia al lenguaje de marcado para la elaboración de páginas web.

El Primer Explorador

Mosaic fue el segundo navegador gráfico disponible para visualizar páginas web y el primer navegador gráfico para Microsoft Windows.

Primer Sitio Web

Fue creado por Tim Berners-Lee usando un computador NeXT, y su función era informar sobre la nueva y emocionante World Wide Web.

1993-1995 Diseño web Con Tablas

Con la llegada de Mosaic en 1993 se consiguen los principales logros gráficos a través de imágenes que se incorporaron. Además, se acerca al usuario común el uso del internet y los navegadores. Aparecen los primeros indicios de Usabilidad e Interface. Se visualizan y distinguen las estructuras básicas de un sitio: Menú, buscador, contenido y footer.

1995-1999 Nuevas tecnologías en Diseño Web

el nacimiento del reciente estándar del lenguaje HTML, hacía que las posibilidades gráficas o multimediales de las páginas web fueran muy limitadas. Existían pocas etiquetas HTML que sólo permitían trabajar con texto e hipertexto, entre otros.La gráfica de esta época corresponde sólo a texto. Los botones están hechos con hipervínculos.

Flash

Flash permitió a los diseñadores y realizadores de páginas web crear entornos gráficos de páginas webs vectoriales e interactivos como por ejemplo: botones, paneles de navegación, dibujos técnicos, banners de anuncios y todo lo que sabemos que se puede hacer con flash.

CSS

CSS se utiliza para dar estilo a documentos HTML y XML, separando el contenido de la presentación. Los Estilos definen la forma de mostrar los elementos HTML y XML. CSS permite a los desarrolladores Web controlar el estilo y el formato de múltiples páginas Web al mismo tiempo.

Javascript

Javascript es un lenguaje con muchas posibilidades, utilizado para crear pequeños programas que luego son insertados en una página web y en programas más grandes, orientados a objetos mucho más complejos. Podemos crear diferentes efectos e interactuar con nuestros usuarios.

Aunque Flash ni CSS eran utilizados masivamente aún debido a que pocos navegadores soportaban el estándar, permitió dar los primeros pasos para comenzar a ser utilizado de manera masiva más adelante. Recordemos que en esta época la velocidad de banda era apenas 28k/sg.

1999-2005 El ascenso de Flash y Javascript

En esta época se concretan y masifican tecnologías que ya habían aparecido pero que faltaban por optimizar o implementar en todos los navegadores.

En el 2000, Internet Explorer se convierte en el primer explorador en soportar perfectamente los estándares de HTML, CSS y Javascprit, por lo cual su difusión entre los usuarios del mundo se hizo evidente, permitiendo que sitios que usaban esta tecnología pudieran ser visualizados por casi cualquier persona en computadores que tuvieran IE.

El boom de Macromedia Flash

En esta década Flash se hizo más popular gracias al aumento de sus características y se convirtió en un plugin estándar incluido en casi todos los navegadores Web. Con esto permitió a los diseñadores y realizadores de páginas web crear entornos gráficos de páginas webs vectoriales e interactivos como por ejemplo: botones, paneles de navegación, dibujos técnicos, banners de anuncios y todo lo que sabemos que se puede hacer con flash.

Esto creó un nuevo elemento que antes no estaba considerado en la elaboración de sitios web, y es la Experiencia de Usuario (UX). Ahora, aparte de considerar una estrategia basada en la usabilidad y arquitectura de la información, también un elemento a considerar era en cómo hacer sentir al usuario al navegar en un sitio web. Esto cobró realmente mucha importancia, pues cada vez era más soprendente navegar por internet.

Javascript

Javascript permitió darle inteligencia a las páginas web. Con JS se podían hacer menús dinámicos y animaciones sin la necesidad de utilizar Flash, lo que permitió rápidamente que se convirtiera en una herramienta utilizada por diseñadores y desarrolladores.

PHP

Con la aparición del lenguaje PHP, la administración de bases de datos pronto sería un elemento indispensable para la administración de los sitios web.

Mail

El correo personal o e-mail se vuelve una herramienta que todo usuario de internet debe poseer. Compañías como Yahoo y Hotmail se vuelven pioneras en este área. Si bien, el correo electrónico antecede a internet, en esta época es donde tendrá mayor protagonismo.

2005-2009 Surgimiento de la Web 2.0

Web 2.0 o Internet 2.0 es una denominación bastante optimista, que da la impresión de un internet totalmente nuevo, interactivo, colaborativo, democrático y sociable frente a todo lo que había sido antes: estático y pasivo. Y este periodo así lo fue. Tim Berners-Lee ha dicho que esta fue su idea de la web desde el principio.

Aparecen en la escena redes sociales tales como MySpace, Facebook, Twitter, Youtube, entre otras, que permiten al usuario, como nunca antes, agregar contenido propio, compartir en tiempo real e interactuar con otros usuarios y ser creadores del contenido que antes era estático.

Los celulares pronto serán teléfonos inteligentes y tendrán un importante papel en el despegue masivo de estas redes sociales. Al principio de esta época, si bien el uso de Flash sigue teniendo importancia, comienza a perderla debido al auge de CSS y el Javascript.

2009-Actualidad: La Web Digital

La gráfica cambia, se alínea y se encamina a un fin común: Usabilidad, Experiencia de Usuario, Arquitectura de la Información y la Interfaz de Usuario. Estos elementos convergen en todos los sitios actuales, donde la importancia de cómo y dónde colocar un simple elemento en una página web puede influye.

Aparecen gráficas completamente vectoriales de colores planos, botones sutiles y enormes, fondos planos y colores poco saturados. La diagramación se vuelve más libre y se juega con todo el ancho de las pantallas. Este cambio gráfico responde a la necesidad de optimizar la visualización de las páginas que visitamos en escritorio a celulares.

La manera de comunicar gráficamente ya no es estática. La creación de un sitio web se vuelve poco a poco en un relato, en algo multimedial e incluso tiene algo de audiovisual. Hay sitios que para implementarse necesitan storyboards que expliquen el flujo y la manera en cómo van a ser visualizados en los navegadores.
